🔌 เชื่อมต่อกับ OpenClaw
1
ใส่ API Key อย่างน้อย 1 ตัว
ไปที่แท็บ "🔑 ใส่ API Key" แล้วใส่ Key อย่างน้อย 1 ตัว (แนะนำ Groq)
2
RouterAI Proxy ต้องกำลังรันอยู่
Proxy ที่ port 8900 ต้องทำงานอยู่ — รัน: python src/proxy.py
3
กดปุ่มเชื่อมต่ออัตโนมัติ
กดปุ่มด้านล่าง ระบบจะตั้งค่า OpenClaw ให้อัตโนมัติ

กดปุ่มนี้ → ระบบจะแก้ ~/.openclaw/openclaw.json ให้อัตโนมัติ

📋 หรือตั้งค่าเอง (ถ้าปุ่มไม่ทำงาน)

คัดลอกโค้ดด้านล่าง แล้ววางใน ~/.openclaw/openclaw.json

{ "llm": { "provider": "openai", "baseUrl": "http://127.0.0.1:8900/v1", "apiKey": "routerai", "model": "groq/llama-3.3-70b-versatile" } }
🔄 รีสตาร์ท OpenClaw

หลังจากตั้งค่าแล้ว ต้องรีสตาร์ท OpenClaw:

openclaw restart
